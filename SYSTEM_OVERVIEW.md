## 🎯 How to Use

### 1. **Start the Application**
```bash
# Activate virtual environment
.\venv\Scripts\activate

# Run the advanced application
python app.py
```

### 2. **Access the Interface**
- Open browser to `http://localhost:5000`
- Beautiful gradient interface loads

### 3. **Upload and Analyze**
- Click "Choose Image File" button
- Select PNG, JPG, or JPEG image
- Click "Analyze Image" button
- View comprehensive results

### 4. **Explore Results**
- Check prediction result (Real/Fake)
- View confidence scores and percentages
- Explore detailed feature analysis
- Examine interactive charts and graphs
