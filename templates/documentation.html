Getting Started

Welcome to the AI Deepfake Detector! This powerful system uses advanced neural networks to detect deepfake and manipulated images with high accuracy.

System Requirements: Python 3.8+, 4GB RAM, Modern web browser

Installation

1
Run the Project

Simply run the main project file:

python project.py

That's it! The system will automatically:

  • Set up the virtual environment
  • Install all dependencies
  • Create necessary directories
  • Start the web application
  • Open your browser to the interface

How to Use

Supported File Formats

  • PNG - Portable Network Graphics
  • JPG/JPEG - Joint Photographic Experts Group
  • Maximum file size: 10MB

Understanding Results

Real Image: The AI determined the image is authentic with high confidence
Fake Image: The AI detected signs of artificial generation or deepfake manipulation

Confidence Scores

  • 90-100%: Very high confidence
  • 70-89%: High confidence
  • 50-69%: Moderate confidence
  • Below 50%: Low confidence - manual review recommended

Editing Detection

The system also detects if an image has been edited or manipulated, even if it's not a deepfake:

  • Compression artifacts - JPEG compression inconsistencies
  • Noise patterns - Unnatural noise distribution
  • Edge inconsistencies - Irregular edge patterns

Troubleshooting

Common Issues

File Upload Error: Ensure your image is under 10MB and in PNG/JPG format
Slow Processing: Large images may take longer to process. Consider resizing to 1024x1024 or smaller
Unexpected Results: Very low quality or heavily compressed images may produce less accurate results

Performance Tips

  • Use high-quality images for best results
  • Avoid heavily compressed or low-resolution images
  • Images with clear faces work best
  • Good lighting improves detection accuracy
